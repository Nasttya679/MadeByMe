# MadeByMe

**MadeByMe** — це веб-платформа для купівлі та продажу саморобних (handmade) продуктів, яка об'єднує продавців унікальних виробів та покупців, зацікавлених у таких товарах.

> 📄 Повний огляд проекту: [docs/Made By Me presentation.pdf](docs/Made%20By%20Me%20presentation.pdf)

---

## 🎯 Мета проекту

Надати єдиний простір для українських майстрів, де вони можуть реалізовувати свої творчі ідеї та продавати вироби без необхідності самостійно займатися маркетингом. Платформа орієнтована на початківців, молодь та всіх, хто хоче монетизувати своє хобі.

---

## 🧩 Основні функції

### Для користувачів (гостей та зареєстрованих)
- Перегляд каталогу товарів із фільтрацією за категоріями та сортуванням
- Перегляд детальних сторінок товарів (фото, опис, характеристики)
- Реєстрація та авторизація (email, соціальні мережі, Google)
- Перегляд і редагування власного профілю
- Написання відгуків до товарів

### Для покупців
- Додавання товарів до кошика
- Оформлення та оплата замовлень
- Перегляд історії замовлень

### Для продавців
- Створення оголошень (назва, опис, ціна, фото)
- Редагування та видалення власних товарів
- Управління статусами замовлень (наприклад, «Відправлено»)
- Перегляд журналу замовлень

### Для модераторів
- Перевірка контенту та видалення порушень (товарів, відгуків)
- Блокування та розблокування користувачів

## 👥 Ролі користувачів

| Роль | Опис |
|---|---|
| **Гість** | Відвідує сайт без реєстрації. Може переглядати товари та здійснювати покупки. |
| **Авторизований користувач** | Зареєстрований акаунт. Може виставляти товари на продаж і залишати коментарі. |
| **Покупець** | Купує вироби, керує кошиком, залишає відгуки. |
| **Продавець** | Створює й керує оголошеннями, продає товари. |
| **Модератор** | Стежить за дотриманням правил спільноти, контролює якість контенту. |

## ⚙️ Нефункціональні вимоги

- **Продуктивність**: час відгуку сторінки до 2 секунд при 1 000 одночасних користувачах
- **Масштабованість**: підтримка до 10 000 одночасних користувачів
- **Безпека**: паролі зберігаються у хешованому вигляді, HTTPS для всіх запитів
- **Надійність**: час безперебійної роботи 99,9% (SLA), автоматичне резервне копіювання кожні 24 години
- **Сумісність**: підтримка браузерів Chrome, Firefox, Safari, Edge
- **Зручність**: інтуїтивний інтерфейс; будь-яка основна дія — не більше 3 кліків
- **Відповідність**: GDPR та законодавство України щодо онлайн-торгівлі

---

## 🏆 Конкурентні переваги

- **Фокус на початківців** — орієнтовано на молодь і людей без досвіду в маркетингу
- **Спрощений шлях** — мінімальний процес додавання товарів: фото + опис + ціна
- **Національна спільнота** — єдиний простір для українських майстрів з активною модерацією
- **Доступність** — інтерфейс для користувачів будь-якого віку та рівня технічних навичок
